# Guía de Instalación

En esta sección te explicamos cómo preparar tu entorno de trabajo para poder editar y compilar esta documentación en tu propio equipo.

## Requisitos previos

Para poder trabajar con este proyecto, necesitarás tener instalados los siguientes componentes:

* **Python** (versión 3.7 o superior).
  
* **Git** para el control de versiones.
* Un editor de código, como **Visual Studio Code**.

## Pasos de instalación local

1. **Clonar el repositorio**: Descarga el código fuente en tu equipo.
2. **Crear un entorno virtual**: Aísla las dependencias del proyecto usando `python -m venv venv`.
3. **Instalar dependencias**: Instala MkDocs y el tema Material ejecutando `pip install mkdocs mkdocs-material`.
